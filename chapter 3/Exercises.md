# 8a

## Weighted Means

$$\bar{X}_w = \frac{\sum w_i X_i}{\sum w_i} = \frac{1(10)+2(12)+3(14)+4(16)}{10} = \frac{140}{10} = 14$$

$$\bar{Y}_w = \frac{\sum w_i Y_i}{\sum w_i} = \frac{1(40)+2(45)+3(40)+4(55)}{10} = \frac{470}{10} = 47$$

## Weighted Slope

$$\hat{\beta}_1 = \frac{\sum w_i(X_i - \bar{X}_w)(Y_i - \bar{Y}_w)}{\sum w_i(X_i - \bar{X}_w)^2} = \frac{100}{40} = 2.5$$

## Weighted Intercept

$$\hat{\beta}_0 = \bar{Y}_w - \hat{\beta}_1 \bar{X}_w = 47 - 2.5(14) = 12$$
